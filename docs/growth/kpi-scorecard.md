# KPI Scorecard for Demand Validation

## Acquisition

- Organic sessions (weekly)
- New users from SEO pages
- Top entry page CTR from search results

## Activation

- Builder start rate
- Builder completion rate
- Median time to first resume draft

## Engagement

- Resume tool usage rate (ATS checker / analyzer / optimizer)
- Return rate (7-day and 30-day)
- Example-page to builder click-through rate

## Validation thresholds before monetization

- >= 5,000 MAU
- >= 25% completion rate
- >= 15% export success rate
- >= 10% returning users
- >= 30% traffic share from organic channels
