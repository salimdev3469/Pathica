# First 3-Month Growth Plan (Pathica)

## Month 1: Foundation and conversion baseline

- Ship high-intent SEO landing pages at core resume slugs.
- Improve homepage with trust signals and tool discovery links.
- Launch blog hub and first wave of indexable guides.
- Track funnel events: landing_visit, builder_start, builder_complete, export_success.

Targets:
- 1,200 MAU
- >= 20% builder start rate from landing
- >= 12% export success rate

## Month 2: SEO expansion and share loops

- Publish full set of role-based example pages and blog content.
- Launch ATS checker, resume analyzer, score, and keyword optimizer tools.
- Launch public template gallery with share link copy flow.
- Add internal linking between SEO pages, tools, and builder.

Targets:
- 2,800 MAU
- >= 25% builder completion rate
- >= 8% users using at least one tool page

## Month 3: Validation and growth acceleration

- Improve top pages by Search Console query data.
- Expand blog cluster for high-impression keywords.
- Add lightweight social sharing hooks and creator-friendly assets.
- Tighten onboarding to reduce time-to-first-value.

Targets:
- 5,000 MAU
- >= 15% export success rate
- >= 10% returning users (7-day)
